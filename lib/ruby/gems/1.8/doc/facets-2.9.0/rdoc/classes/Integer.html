Public Instance methods

bit(bit)

Set a bit. 

  0.bit(4)  #=> 16

Using a negative figure will clear a bit. 

  10.bit(-4)      #=> 2

This is more easily seen using binary. 

  0b0100.bit(-3)  #=> 0

CREDIT: Thomas Sawyer, George Moschovitis

bit?(bit)

Is a bit set? 

  8.bit?(3)  #=> true
  8.bit?(2)  #=> false

CREDIT: Thomas Sawyer, George Moschovitis

bit_clear(bit)

Clear bit.

CREDIT: George Moschovitis

bitmask(mask)

Apply a bitmask. 

  1.bitmask(6) #=> 7

Using a inverted bitmask clears bits. 

  7.bitmask(~2) #=> 5
  5.bitmask(~2) #=> 5

CREDIT: George Moschovitis

bitmask?(mask)

Is bitmask set? 

  7.bitmask?(7) #=> true
  7.bitmask?(5) #=> true
  8.bitmask?(3) #=> false

CREDIT: George Moschovitis

Returns true if this integer is odd, false otherwise.
