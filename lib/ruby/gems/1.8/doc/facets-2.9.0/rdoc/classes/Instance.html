update(hash)

Set instance variables given a hash. 

  instance.update('@a'=>1, '@b'=>2)
  @a   #=> 1
  @b   #=> 2

Also, +@+ sign is not neccessary. 

  instance.update(:a=>1, :b=>2)
  @a   #=> 1
  @b   #=> 2
